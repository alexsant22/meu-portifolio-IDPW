# Portfólio Profissional - Alexandre Santos

Este é um projeto de portfólio moderno e responsivo construído exclusivamente com tecnologias web fundamentais. O site foi desenvolvido como trabalho prático para a disciplina de **Introdução ao Desenvolvimento de Páginas Web (ARQIDPW)**, servindo como uma vitrine profissional para recrutadores e desenvolvedores.

## 🚀 Funcionalidades

- **Navegação Semântica**: Estrutura baseada em HTML5 puro para melhor indexação e acessibilidade.
- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops utilizando CSS Flexbox e Media Queries.
- **Acessibilidade**: Implementação de VLibras, textos alternativos em imagens e alto contraste.
- **Gestão de Contato**: Formulário estruturado com validações nativas do HTML5.
- **Navegação Interna**: Uso de Bookmarks (âncoras) para facilitar a leitura de projetos longos.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica de conteúdo.
- **CSS3**: Estilização avançada e layout responsivo (Vanilla CSS).
- **Metodologia**: Mobile-first design.
- _(Nota: Por requisitos acadêmicos, este projeto não utiliza frameworks ou JavaScript)._

## 📋 Requisitos Acadêmicos Atendidos

O projeto cumpre integralmente os seguintes critérios técnicos:

- **Multimídia**: Inclusão de vídeo de apresentação na página "Sobre".
- **Tabelas**: Comparativo técnico de competências e níveis de proficiência.
- **Listas**: Organização de soft skills e tecnologias.
- **Formulário**: Campos de texto, e-mail, seleção e botões de ação (Enviar/Limpar).
- **Links**: Integração com redes externas (GitHub/LinkedIn) e links internos.

## 📂 Estrutura de Pastas

A arquitetura do diretório foi organizada para garantir escalabilidade e fácil manutenção:

```text
meu-portifolio-IDPW/
│
├── index.html               # Home: Apresentação e resumo principal
├── sobre.html               # Bio: Trajetória e conteúdo multimídia
├── habilidades.html         # Skills: Listas e tabelas técnicas
├── projetos.html            # Portfolio: Cards e bookmarks internos
├── contato.html             # Contact: Formulário de mensagem
│
└── assets/                  # Recursos globais
    ├── css/
    │   └── style.css        # Folha de estilo única
    ├── img/                 # Assets de imagem (perfil, projetos)
    └── media/               # Arquivos de vídeo e áudio
```

## 💻 Páginas do Projeto

1.  **Página Inicial**: Primeiro contato com o usuário, contendo foto de perfil e resumo executivo.
2.  **Sobre Mim**: Detalhamento da carreira com um vídeo de introdução integrado.
3.  **Habilidades**: Exibição técnica utilizando tabelas para comparar níveis de conhecimento (Iniciante ao Avançado).
4.  **Projetos**: Divisão entre projetos Front-end e Back-end acessíveis via menu de âncoras.
5.  **Contato**: Interface para envio de mensagens com validação de campos obrigatórios.

## ⚙️ Como Visualizar

Para executar o projeto localmente, não é necessário um ambiente de build:

1. **Clonagem do repositório:**

```bash
git clone https://github.com/alexsant22/meu-portifolio-IDPW.git
```

2. **Execução:**
   Basta abrir o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge).
